LIBRARY

Overview:
-The Book Library Website is a simple web application that allows users to add, view, and manage a collection of books. This project is designed to help beginners understand the basics of web development, including HTML, CSS, and JavaScript.

Features:
-Add new books to the library
-View the list of books
-Delete books from the library
-Responsive design for better user experience on different devices

Technologies Used:
-HTML
-CSS
-JavaScript

Getting Started

Prerequisites:
-To run this project, you will need a web browser like Chrome, Firefox, or Edge.

Installation:
Clone the repository:
git clone https://github.com/yourusername/library.git

Navigate to the project directory:
cd book-library-website

Open the index.html file in your web browser:

open index.html
or
start index.html

Usage: 
-Open the website in your web browser.
-To add a new book, fill in the "Title", "Author", and "pages" fields and click the "Submit" button.
-To delete a book, click the "Delete" button next to the book you want to remove.
-Moreover,we can also add the opton that whether we have read the book or not.
